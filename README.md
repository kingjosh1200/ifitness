# iFitness Web Architecture Study
### Tech Stack: HTML5 | CSS3 | Bootstrap 5 | JavaScript (jQuery)

## 📌 Project Overview
This is a multi-page responsive website project designed to recreate and study the interface of the **iFitness** platform. The goal of this project was to master the **Bootstrap 5 grid system** and integrate third-party JavaScript libraries to create a professional-grade user experience.

## 📂 Project Structure & Files
- **`index.html`**: The main landing page featuring the hero section and core brand messaging.
- **`about.html` & `corporate.html`**: Information architecture for business values and B2B services.
- **`membership.html`**: Comparison tables for different fitness subscription tiers.
- **`shop.html` & `shop2.html`**: E-commerce layouts designed for fitness apparel and equipment.
- **`style.css`**: Custom CSS used to override Bootstrap defaults and match brand-specific aesthetics.
- **`images/`**: Organized directory for all visual assets.
- **`.vscode/`**: Local development environment configurations.

## 🚀 Technical Features
- **Library Integration:** Utilized `lightbox-plus-jquery.min.js` and `lightbox.min.css` to implement interactive image galleries for gym facilities.
- **Responsive Design:** Used Bootstrap breakpoints to ensure the interface is fully functional on mobile, tablet, and desktop.
- **Navigation:** Implemented a consistent, user-friendly navigation system across 7 distinct HTML pages.

## 🛠️ How to View
1. Clone this repository.
2. Open `index.html` in any modern browser.
